# <div style='color: #ffbbbb'>间断点

## 例题
<div style='color: #1177bb'>

> 函数 $f(x)=\frac{1-\cos |x|}{|x| \ln |x+1|}$ 的跳跃间断点的个数为

</div>

    A. 0

    B. 1

    C. 2

    D. 3 



<details>
<summary style='color: pink'>点击查看答案</summary>

<h2>

C

</h2>

1. 先找间断点，即函数中无定义的点:

   * 包含$\ln |x+1|$ 与分式。

1. $\ln |x+1|$, 在 $x=-1$ 处无定义。

1. $|x| \ln |x+1|=0$ $\rightrightarrows$ $x=0, x=-2$. 综上 $x=0,-1,-2$ 是 $f(x)$ 的间断点。
判断间断点的类型。
1. 考虑 $x=0$.
由于绝对值, 分别讨论左极限和右极限。
